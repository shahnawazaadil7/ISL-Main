    We create accessible tools and resources to simplify learning Indian Sign Language, fostering communication and inclusivity for all.

    How We Make a Difference!

    STEP 1 :



    Detect: The system uses computer vision (like MediaPipe) to detect hand landmarks or movements from live video.

    STEP 2 :



    Predict: A trained machine learning model (e.g., LSTM) processes the detected landmarks and predicts the corresponding Indian Sign Language (ISL) letter or word.

    STEP 3 :



    Display: The predicted text is then shown on the screen, translating ISL signs into readable text.
